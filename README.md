# 🍌 Casa Madu — Buñuelitos de Maduro PWA

App instalable (PWA) para pedidos de buñuelitos de maduro artesanales.

## Archivos del proyecto

```
casa-madu/
├── index.html       ← App principal
├── manifest.json    ← Config PWA
├── sw.js            ← Service Worker (offline)
├── icons/
│   ├── icon-192.png
│   └── icon-512.png
└── README.md
```

## Pasos para publicar

### Opción A — Netlify (recomendado)
1. Crea carpeta `casa-madu/` en tu PC con todos los archivos
2. Agrega los iconos en `icons/` (192×192 y 512×512 PNG)
3. Arrastra la carpeta a https://app.netlify.com/drop
4. ¡Listo! — La URL quedará tipo `casamadu.netlify.app`

### Opción B — GitHub Pages
1. Crea repo `casamadu` en haroldco45
2. Sube todos los archivos (root del repo)
3. Settings → Pages → Source: main / root
4. URL: `haroldco45.github.io/casamadu`

## Iconos (pendiente)
Crea o exporta un ícono de 🍌 en fondo verde `#2D5016`:
- `icons/icon-192.png` (192×192 px)
- `icons/icon-512.png` (512×512 px)

Puedes usar https://realfavicongenerator.net
